UK Postcode Heat Mapper

Paste postcodes directly, optionally with dates. Filter across multiple date periods, assign separate colours, and export the finished map as a PNG. This is a fully static front-end app: no backend, no database, no paid service.

1) Paste your data

Accepted examples:
BR8 7RE
2026-03-01, BR8 7RE
BR8 7RE, 01/03/2026
Job123, BR8 7RE, 2026-03-01

2) Date periods and colours

Add one or more layers. Each layer is filtered by date range and drawn in its own colour. If your data has no dates, create a single layer and leave dates blank.

3) Notes

• Full postcode locations are resolved in the browser using the free Postcodes.io API and then cached locally for re-use.
• This means the app stays static and GitHub Pages-friendly.
• Invalid or unresolved postcodes are listed in the summary so they can be cleaned up.
Tip: for a Windows desktop feel, you can open this locally or host it on GitHub Pages and install it as an app in Edge or Chrome.
